# 🏠 Wright Family Home

A Nigerian family home management app built with React. Manage chores, meals, groceries, routines, and family dates — all in one place.

## Features
- 🏠 Home dashboard with daily summary
- 🧹 Chore manager with member assignments
- 🛒 Grocery shopping list
- 📅 Family dates & events calendar
- 🍲 Weekly meal planner with Nigerian dishes
- ⚡ Daily routines for each family member

## Tech Stack
- React 18
- Vite
- PWA-ready (installable on phone)

## Deploy on Vercel
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) and import the repo
3. Click Deploy — done!

## Run Locally
```bash
npm install
npm run dev
```
